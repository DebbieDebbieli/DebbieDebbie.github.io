Markdown语法

Nov 30, 2018

1.Markdown简介

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，可使普通文本内容具有一定的格式。

基于Markdown的简介性，它已经成为目前世界上最流行的书写博客的语言。

2.Markdown语法

本文采用的是Notepad++书写，安装了MarkdownViewerPlusPlus插件，配置了Mrakdown语法高亮的.xml文件。

MarkdownViewerPlusPlus可以在Notepad++的
“插件”->“Plugin Manager”
中安装。

语法高亮插件在Github上有链接地址：
https://github.com/Edditoria/markdown-plus-plus

问题：在本文还没有写完时，发现win10下经常卡死，于是换了vscode。

(1)标题

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

(2)粗体、斜线、删除线和下划线

*斜体*

**粗体**

***加粗斜体***

~~删除线~~

(3)超链接

[链接文字](链接地址 “链接描述”)

例如：[我的博客](https://debbiedebbieli.github.io/ “我的博客”)

<链接地址>

例如：<https://debbiedebbieli.github.io/>

(4)引用

> 文字引用

(5)代码

a单行代码a

aaa

多行代码

多行代码

aaa

(6)分割线

-–

***

(7)列表

* 列表项

- 列表项

+ 列表项

以上三种方法均可表示列表

(8)表格

表头1|表头2

-|-|-

内容11|内容12

内容21|内容22

(9)图片

![图片文字] (图片地址 “图片描述”)

例如：

![图片文字] (C:\Users\lbb\Desktop\xiaocai\IMG_0326.JPG “图片描述”)

说明：在Hexo中插入图片，按照以下步骤：

I.将站点配置文件中的post_asset_folder设置为true

II.在站点文件夹中打开git bash，输入命令

npm install hexo-asset-image --save

III.在source/_post文件夹中生成一个以文章名命名的文件夹，将图片放到这里即可。引用方式：

!示例图片 (title/example.png “示例图片”)

(10)转义字符

一般转义和C语言中差不多，用‘\’即可，

字符 转义表示
空格 &nbsp;
< &lt;
> &gt;

(11)字体、文字颜色

<font face=”字体”,color=#DC143C>文字</font>
